### Описание
Приложение для командной строки, которое определяет "похожесть" изображений.

Данные вводятся через `std::cin`, выводятся через `std::cout`.

Входные данные:
- первая строка - пороговое значение "похожести" в процентах,
- остальные строки - пути к файлам картинок, количество файлов произвольное.

Выходные данные:
 - пары картинок, которые "похожи" друг на друга больше порогового значения, с указанием значения "похожести".

Программа написана на C++ с использованием OpenCV, проект MS Visual Studio 2019.

В качестве критерия "похожести" выбрано взвешенное среднее арифметическое между корреляцией гистограмм RGB каналов изображений и 
коэффициентом, вычисленным как 1 минус среднее арифметическое расстояний сопоставленных дескрипторов, отнормированных к 1. Дескрипторы 
изображений находятся по алгоритму ORB из OpenCV. Таким образом 100% соответствует одинаковым изображениям.
